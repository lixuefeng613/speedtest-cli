# speedtest-cli

下载：wget   https://gist.githubusercontent.com/lixuefeng613/41507a21e5ed2a4652f227097e18ca0a/raw/c81f9d563e153e6919fc796aa654aabd952eb36f/speedtest-cli.sh

赋予权限：chmod +x speedtest-cli.sh

执行：sh speedtest-cli.sh
